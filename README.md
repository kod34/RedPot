# RedPot

# Description

A Web + SSH + MySQL HoneyPot designed to be setup and run on a virtual machine.

# Warning

Installing this on your personal machine might lead you to lose your data.

# Installation 

chmod +x install.sh  
sudo ./install.sh

# Setup environment

chmod +x setup.sh  
sudo ./setup.sh

# Run Redpot

chmod +x redpot.sh  
sudo ./redpot
